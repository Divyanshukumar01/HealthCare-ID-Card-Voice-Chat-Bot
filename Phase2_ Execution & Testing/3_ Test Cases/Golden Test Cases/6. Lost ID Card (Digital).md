# 6. Lost ID Card (Digital)

---

# Objective

| Item | Description |
|------|-------------|
| Test Objective | Verify that the Healthcare ID Card Voice Assistant successfully processes a lost digital healthcare ID card replacement request for an authenticated member. |
| Intent Tested | Lost Digital ID Card Replacement |
| Authentication Required | Yes |
| Primary Agent | Root Agent |
| Secondary Agent | Lost Card Replacement Agent |
| Expected Result | Digital replacement card request is submitted successfully and the previous card is deactivated. |

---

# Test Scenario

| Field | Value |
|------|-------|
| Scenario Name | Lost ID Card (Digital) |
| User Type | Existing Member |
| Intent | Replace Lost Healthcare ID Card |
| Authentication Required | Yes |
| Specialized Agent | Lost Card Replacement Agent |
| Expected Result | Digital replacement request is created successfully. |

---

# Conversation Flow

| Step | User Input | Assistant Action | Expected Result |
|------|------------|------------------|-----------------|
| 1 | hi | Greets the user. | Greeting is provided successfully. |
| 2 | i lost my id card | Identifies the replacement request. | Confirms the lost ID card intent. |
| 3 | yes | Requests member verification. | Authentication process begins. |
| 4 | m001 | Requests Date of Birth. | Member identification continues. |
| 5 | 1995-05-10 | Validates member details and retrieves profile information. | Member is authenticated successfully. |
| 6 | — | Transfers conversation to the Lost Card Replacement Agent. | Specialized agent handles the replacement request. |
| 7 | lost | Confirms the reason for replacement and explains that the existing card will be deactivated. | User is informed about card deactivation. |
| 8 | digital | Collects preferred replacement card type. | Digital replacement preference is confirmed. |
| 9 | yes | Executes the **replaceLostCard** tool. | Replacement request is submitted successfully. |
| 10 | no | Ends the conversation. | Session is completed successfully. |

---

# Tool Execution Summary

| Tool | Purpose | Expected Response |
|------|---------|------------------|
| getCurrentDate | Retrieves the current system date and time. | Current date and time returned successfully. |
| validateMember | Authenticates the member using Member ID and Date of Birth. | Member is authenticated successfully. |
| getMemberDetails | Retrieves member profile information. | Member details are returned successfully. |
| replaceLostCard | Creates a replacement card request and deactivates the previous card. | Replacement request ID, delivery timeline, and previous card status are returned. |
| end_session | Ends the conversation after successful completion. | Session is terminated successfully. |
