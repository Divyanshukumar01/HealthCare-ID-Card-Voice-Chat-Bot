# 4. Access Issue


| Field | Description |
|-------|-------------|
| **Test Case Name** | Access Issue |
| **Purpose** | Verify that the Healthcare ID Card Voice Assistant correctly handles a member's healthcare ID card access issue. |
| **Description** | The assistant authenticates the member, retrieves member details, transfers the conversation to the Access Issue Agent, collects the issue details, creates a support ticket, shares the ticket number with the member, and closes the conversation successfully. |

---

# Objective

| Objective ID | Validation |
|-------------|------------|
| OBJ-1 | Detect the Access Issue intent. |
| OBJ-2 | Authenticate the existing member. |
| OBJ-3 | Retrieve member details successfully. |
| OBJ-4 | Transfer the conversation to the Access Issue Agent. |
| OBJ-5 | Collect complete issue information from the member. |
| OBJ-6 | Create a support ticket using the backend tool. |
| OBJ-7 | Share the ticket number with the member. |
| OBJ-8 | End the conversation successfully. |

---

# Test Scenario

| Field | Value |
|-------|-------|
| **Scenario Name** | Access Issue |
| **Display Name** | access issue |
| **User Type** | Existing Member |
| **Intent** | Report Healthcare ID Card Access Issue |
| **Authentication Required** | Yes |
| **Authentication Method** | Member ID + Date of Birth |
| **Specialized Agent** | Access Issue Agent |
| **Backend Tools Used** | getCurrentDate, validateMember, getMemberDetails, raiseAccessIssue, end_session |
| **Expected Result** | Support ticket created successfully |

# Conversation Flow

| Step | System Action |
|------|---------------|
| 1 | Greets the member. |
| 2 | Detects Access Issue intent. |
| 3 | Requests Member ID. |
| 4 | Requests Date of Birth. |
| 5 | Authenticates the member. |
| 6 | Retrieves member information. |
| 7 | Transfers the conversation to the Access Issue Agent. |
| 8 | Collects issue description. |
| 9 | Confirms the issue details. |
| 10 | Creates a support ticket. |
| 11 | Shares the ticket number with the member. |
| 12 | Ends the conversation politely. |
