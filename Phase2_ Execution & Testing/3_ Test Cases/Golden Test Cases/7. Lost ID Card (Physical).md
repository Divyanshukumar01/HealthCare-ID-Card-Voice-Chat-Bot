# 7. Lost ID Card (Physical)

---

# Objective

| Item | Description |
|------|-------------|
| Test Objective | Verify that the assistant successfully processes a lost physical healthcare ID card replacement request for an authenticated member. |
| Intent Tested | Lost Physical ID Card Replacement |
| Authentication Required | Yes |
| Primary Agent | Root Agent |
| Secondary Agent | Lost Card Replacement Agent |
| Expected Result | Physical replacement card request is submitted successfully and the previous card is deactivated. |

---

# Test Scenario

| Field | Value |
|------|-------|
| Scenario Name | Lost ID Card (Physical) |
| User Type | Existing Member |
| Initial Intent | Replace Lost Healthcare ID Card |
| Authentication | Required |
| Transfer Required | Yes |
| Final Outcome | Physical replacement request submitted successfully |

---

# Conversation Flow

| Step | User Input | Assistant Action | Expected Result |
|------|------------|------------------|-----------------|
| 1 | hi | Greets the user. | Welcome message is provided. |
| 2 | i lost my id card | Identifies lost card request. | Confirms replacement request. |
| 3 | yes | Starts identity verification. | Requests Member ID or registered email. |
| 4 | m001 | Accepts Member ID. | Requests Date of Birth. |
| 5 | 1995-05-10 | Authenticates member and retrieves profile details. | Member verified successfully. |
| 6 | — | Transfers conversation to Lost Card Replacement Agent. | Specialized agent takes over. |
| 7 | lost | Asks replacement preference after informing previous card will be deactivated. | User chooses replacement type. |
| 8 | physical | Requests delivery address. | Delivery address collected. |
| 9 | jammu 18002 | Confirms replacement details and delivery location. | Awaits final confirmation. |
| 10 | yes | Calls **replaceLostCard** tool. | Physical replacement request submitted successfully. |
| 11 | — | Shares Request ID and expected delivery timeline. | User receives confirmation and tracking details. |
| 12 | no | Calls **end_session** tool. | Conversation ends successfully. |

---

# Tool Execution Summary

| Tool | Purpose | Expected Response |
|------|---------|------------------|
| getCurrentDate | Retrieves the current system date and time. | Date and time returned successfully. |
| validateMember | Authenticates the member using Member ID and DOB. | Member authentication successful. |
| getMemberDetails | Retrieves member profile information. | Member details returned successfully. |
| replaceLostCard | Creates a physical replacement card request and deactivates the previous card. | Request ID generated and replacement initiated. |
| end_session | Ends the conversation after successful request completion. | Session terminated successfully. |
