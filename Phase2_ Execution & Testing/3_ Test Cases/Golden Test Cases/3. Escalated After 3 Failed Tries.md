# 3. Escalated After 3 Failed Tries



| Field | Description |
|-------|-------------|
| **Test Case Name** | Escalated After 3 Failed Tries |
| **Purpose** | Verify that the Healthcare ID Card Voice Assistant correctly handles repeated unrecognized user inputs by escalating the conversation to a live support agent after three failed attempts. |
| **Description** | The assistant attempts to understand the user's request three times. When the intent cannot be identified, it transfers the conversation to the Escalation Agent, initiates live agent handoff, informs the user about the estimated waiting time, and ends the session successfully. |

        text: hi

    - expectation:

          id: adk-3bcb3f8e-fa1e-4b2f-8918-1ed42b90847d
          args: {}
          displayName: Authtool
          toolsetTool:
            toolset: Authtool
            toolId: getCurrentDate

    - expectation:
        toolResponse:
          id: adk-3bcb3f8e-fa1e-4b2f-8918-1ed42b90847d
          response:
            currentDate: '2026-07-07'
            currentTime: 04:57:41
            dayOfWeek: Tuesday
            month: July
            success: true
            timezone: Asia/Kolkata
            year: 2026.0

    - expectation:
        agentResponse:
          role: Root agent
          chunks:
          - text: Hello! Welcome to Healthcare ID Card Services. How may I assist you today?

  - steps:
    - userInput:
        text: eggeg

    - expectation:
        agentResponse:
          role: Root agent
          chunks:
          - text: I'm sorry, I didn't quite catch that. Could you please repeat what you need help with?

  - steps:
    - userInput:
        text: ergreg

    - expectation:
        agentResponse:
          role: Root agent
          chunks:
          - text: I'm still having trouble understanding. Could you please tell me how I can help you today? For example, are you looking to request a new card, replace a lost one, or check the status of a card?

    - userInput:
        text: ergege

    - expectation:
        toolCall:
          id: adk-1a9d0bce-34dc-412a-a7e2-59c5e5ad892d

            reason: member could not be assisted
          displayName: Authtool
          toolsetTool:
            toolset: Authtool
            toolId: escalateToAgent

    - expectation:
        toolResponse:
          id: adk-1a9d0bce-34dc-412a-a7e2-59c5e5ad892d
          response:
            estimatedWait: 5 minutes
            memberId: Unknown
            message: Transferring you to a live agent. Please hold.
            queuePosition: 3.0
            reason: member could not be assisted
            status: Escalated
            success: true

    - expectation:
        agentResponse:
          role: Escalation Agent
          chunks:
          - text: You're now being connected — estimated wait is 5 minutes. Thank you for your patience.

    - expectation:
        toolCall:
          id: adk-4cc3464f-0baa-4e2d-9aa6-73b76a357ee4
          tool: end_session
          args:
            params:
              ESCALATION_MESSAGE: You're now being connected — estimated wait is 5 minutes. Thank you for your patience.
              LIVE_AGENT_HANDOFF:
                endSession: true
            reason: member could not be assisted
            session_escalated: true
