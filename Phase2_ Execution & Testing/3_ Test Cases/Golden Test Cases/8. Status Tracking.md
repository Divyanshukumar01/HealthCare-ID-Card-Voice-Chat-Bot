# 8. Status Tracking

---

# Objective

| Item | Description |
|------|-------------|
| Test Objective | Verify that the assistant successfully tracks the status of an existing healthcare ID card request after authenticating the member. |
| Intent Tested | ID Card Status Tracking |
| Authentication Required | Yes |
| Primary Agent | Root Agent |
| Secondary Agent | ID Card Status Tracking Agent |
| Expected Result | Current ID card request status is retrieved and shared with the member. |

---

# Test Scenario

| Field | Value |
|------|-------|
| Scenario Name | Status Tracking |
| User Type | Existing Member |
| Initial Intent | Check ID Card Status |
| Authentication | Required |
| Transfer Required | Yes |
| Final Outcome | Current request status displayed successfully |

---

# Conversation Flow

| Step | User Input | Assistant Action | Expected Result |
|------|------------|------------------|-----------------|
| 1 | hi | Greets the user. | Welcome message is provided. |
| 2 | i want to check status of my id card | Identifies status tracking request. | Confirms the user's intent. |
| 3 | yes | Starts member verification. | Requests Member ID or registered email. |
| 4 | m001 | Accepts Member ID. | Requests Date of Birth. |
| 5 | 1995-05-10 | Authenticates the member and retrieves profile details. | Member verified successfully. |
| 6 | — | Transfers conversation to ID Card Status Tracking Agent. | Specialized agent takes over. |
| 7 | — | Calls **trackIDCardStatus** tool. | Current request status is retrieved successfully. |
| 8 | — | Shares the current card status with the member. | Status information is communicated. |
| 9 | no | Calls **end_session** tool. | Conversation ends successfully. |

---

# Tool Execution Summary

| Tool | Purpose | Expected Response |
|------|---------|------------------|
| getCurrentDate | Retrieves the current system date and time. | Date and time returned successfully. |
| validateMember | Authenticates the member using Member ID and DOB. | Member authentication successful. |
| getMemberDetails | Retrieves member profile information. | Member details returned successfully. |
| trackIDCardStatus | Retrieves the latest ID card request status. | Current processing status and expected delivery date returned. |
| end_session | Ends the conversation after providing the status. | Session terminated successfully. |
