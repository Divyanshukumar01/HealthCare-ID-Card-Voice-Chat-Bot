# 3. Escalation to Live Agent from Root Agent

---

# Objective

| Item | Description |
|------|-------------|
| Test Objective | Verify that the Healthcare ID Card Voice Assistant transfers the conversation to a live support agent after multiple unsuccessful attempts to understand the user's request. |
| Intent Tested | Live Agent Escalation |
| Authentication Required | No |
| Primary Agent | Root Agent |
| Secondary Agent | Escalation Agent |
| Expected Result | User is successfully connected to a live support representative. |

---

# Test Scenario

| Field | Value |
|------|-------|
| Scenario Name | Escalation to Live Agent from Root Agent |
| User Type | Unknown User |
| Intent | Live Agent Escalation |
| Authentication Required | No |
| Specialized Agent | Escalation Agent |
| Expected Result | Conversation is escalated to a live support agent. |

---

# Conversation Flow

| Step | User Input | Assistant Action | Expected Result |
|------|------------|------------------|-----------------|
| 1 | hi | Greets the user and starts the conversation. | Greeting is provided. |
| 2 | ergr | Unable to recognize the request. | First clarification message is given. |
| 3 | erger | Still unable to understand the request. | Second clarification message is provided. |
| 4 | ergerg | Determines that automated assistance is insufficient. | Conversation is transferred to the Escalation Agent. |
| 5 | — | Escalation Agent informs the user about live support. | User is notified that a live agent will assist. |
| 6 | — | Executes the **escalateToAgent** backend tool. | Escalation request is created successfully. |
| 7 | — | Shares estimated waiting time with the user. | User is informed to wait approximately 5 minutes. |
| 8 | — | Executes the **end_session** tool. | Session ends after successful live-agent handoff. |

---

# Tool Execution Summary

| Tool | Purpose | Expected Response |
|------|---------|------------------|
| getCurrentDate | Retrieves the current system date and time. | Current date and time are returned successfully. |
| escalateToAgent | Creates a live support escalation request. | Queue position and estimated waiting time are returned. |
| end_session | Ends the conversation after escalation. | Session is terminated successfully. |
