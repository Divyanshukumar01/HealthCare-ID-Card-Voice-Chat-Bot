# 9. Informational Query & New Request

---

# Objective

| Item | Description |
|------|-------------|
| Test Objective | Verify that the Healthcare ID Card Voice Assistant can first answer an informational query about available services and then seamlessly process a new Healthcare ID Card request within the same conversation. |
| Intent Tested | Informational Query followed by New Healthcare ID Card Request |
| Authentication Required | Yes |
| Primary Agent | Root Agent |
| Secondary Agent | ID Card Creation Agent |
| Expected Result | User receives service information, completes authentication, and successfully submits a new Healthcare ID Card request. |

| Field | Value |

| User Type | Existing Member |

# Conversation Flow

| Step | User Input | Assistant Action | Expected Result |
|------|------------|------------------|-----------------|
| 1 | What services do you offer? | Explains all Healthcare ID Card services available. | User understands available services. |
| 2 | I need a new ID card. | Identifies the new request and starts the ID card workflow. | New request is initiated. |
| 3 | Existing Member | Determines membership status. | Authentication process begins. |
| 4 | Member ID (M54321) | Requests Date of Birth for verification. | Member verification continues. |
| 5 | Date of Birth (1985-06-20) | Authenticates the member successfully. | Member identity verified. |
| 6 | — | Retrieves member information. | Member profile loaded successfully. |
| 7 | — | Transfers conversation to the ID Card Creation Agent. | Specialized agent handles the request. |
| 8 | Digital/Physical Preference | Creates a new Healthcare ID Card request. | Request submitted successfully. |
| 9 | No | Ends the conversation. | Session completed successfully. |
