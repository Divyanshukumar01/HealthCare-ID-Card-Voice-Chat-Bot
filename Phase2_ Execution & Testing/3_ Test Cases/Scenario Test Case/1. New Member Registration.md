# 1. New Member Registration

---

# Objective

| Item | Description |
|------|-------------|
| Test Objective | Verify that the assistant successfully registers a new member by collecting all required information and creating a new healthcare ID card request. |
| Intent Tested | New Member Registration |
| Authentication Required | No |
| Primary Agent | Root Agent |
| Secondary Agent | Member Registration Agent |
| Expected Result | The new member is registered successfully, a healthcare ID card request is created, and confirmation is provided. |

| Field | Value |
|------|-------|
| Scenario Name | New Member Registration |
| User Type | New Member |
| Initial Intent | Register for a New Healthcare ID Card |
| Authentication | Not Required |
| Transfer Required | Yes |

# Conversation Flow

| Step | User Input | Assistant Action | Expected Result |
|------|------------|------------------|-----------------|
| 1 | Requests to register for a new healthcare ID card. | Identifies the user's intent as a new member registration. | New member registration request recognized successfully. |
| 2 | States that they are a new member. | Initiates the new member registration workflow. | Registration process started successfully. |
| 3 | Provides email (**jane.doe@example.com**). | Validates and stores the email address. | Email accepted successfully. |
| 4 | Provides full name (**Jane Doe**). | Records the member's full name. | Full name captured successfully. |
| 5 | Provides Date of Birth (**1992-11-23**). | Validates the Date of Birth. | Date of Birth accepted successfully. |
| 6 | Provides card preference (**Physical**). | Records the preferred ID card type. | Card preference captured successfully. |
| 7 | Provides mailing address (**123 Main St, Anytown, CA 90210**). | Records the delivery address. | Mailing address captured successfully. |
| 8 | Transfers the conversation to the Member Registration Agent. | Specialized agent processes the registration request. | Agent transfer completed successfully. |
| 9 | Creates the new member profile and healthcare ID card request. | Executes **registerMember** and **createIDCard** tools. | Member registration and ID card request completed successfully. |
| 10 | Shares registration confirmation, Member ID, Request ID, and expected delivery timeline before ending the session. | Provides confirmation details and closes the conversation. | User receives confirmation and the session ends successfully. |

---

# Tool Execution Summary

| Tool | Purpose | Expected Response |
|------|---------|------------------|
| validateEmail | Validates the provided email address. | Email validated successfully. |
| registerMember | Creates a new member profile in the healthcare system. | New Member ID generated successfully. |
| createIDCard | Creates a new healthcare ID card request. | Request ID generated successfully. |
| getCurrentDate | Retrieves the current system date and time. | Current date and time returned successfully. |
| end_session | Ends the conversation after successful registration. | Session terminated successfully. |
