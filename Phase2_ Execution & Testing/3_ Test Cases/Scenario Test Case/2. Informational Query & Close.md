# 2. Informational Query & Close

---

# Objective

| Item | Description |
|------|-------------|
| Test Objective | Verify that the Healthcare ID Card Voice Assistant can accurately answer a user's informational query about available services and gracefully end the conversation when no further assistance is required. |
| Intent Tested | Informational Query |
| Authentication Required | No |
| Primary Agent | Root Agent |
| Secondary Agent | None |
| Expected Result | User receives information about available Healthcare ID Card services and the conversation ends successfully upon the user's request. |

---

# Test Scenario

| Field | Value |
|------|-------|
| Scenario Name | Informational Query & Close |
| User Type | Unknown User |
| Initial Intent | Ask About Available Services |
| Authentication | Not Required |
| Maximum Turns | 10 |
| Final Outcome | Information Provided and Conversation Closed Successfully |

---

# Conversation Flow

| Step | User Input | Assistant Action | Expected Result |
|------|------------|------------------|-----------------|
| 1 | What services do you offer? | Explains the Healthcare ID Card services available to members. | User receives complete service information. |
| 2 | Nothing else is needed. | Acknowledges the user's response and thanks them for contacting Healthcare ID Card Services. | Conversation ends politely. |
| 3 | — | Calls the **end_session** tool. | Session is terminated successfully. |

---

# Tool Execution Summary

| Tool | Purpose | Expected Response |
|------|---------|------------------|
| getCurrentDate | Retrieves the current system date and time. | Date and time returned successfully. |
| end_session | Ends the conversation after the informational request is completed. | Session terminated successfully. |

---

# Agent Transfer
