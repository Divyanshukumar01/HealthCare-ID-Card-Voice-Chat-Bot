# 6. New ID Card – Member ID & Date of Birth Authentication

---

# Objective

| Item | Description |
|------|-------------|
| Test Objective | Verify that the assistant successfully authenticates an existing member using Member ID and Date of Birth before processing a new healthcare ID card request. |
| Intent Tested | New Healthcare ID Card Request |
| Authentication Required | Yes |
| Primary Agent | Root Agent |
| Secondary Agent | ID Card Creation Agent |
| Expected Result | Member is authenticated successfully, and a new healthcare ID card request is submitted successfully. |

# Conversation Flow

| Step | User Input | Assistant Action | Expected Result |
|------|------------|------------------|-----------------|
| 1 | Requests a new healthcare ID card. | Identifies the user's intent. | New ID card request recognized successfully. |
| 2 | States that they are an existing member. | Begins the authentication process. | Authentication initiated successfully. |
| 3 | Provides Member ID (**M12345**). | Requests Date of Birth. | Date of Birth requested successfully. |
| 4 | Provides Date of Birth (**1990-03-15**). | Validates member credentials. | Member authenticated successfully. |
| 5 | Retrieves member profile information. | Fetches member details from the backend system. | Member information returned successfully. |
| 6 | Transfers the conversation to the ID Card Creation Agent. | Specialized agent handles the ID card request. | Agent transfer completed successfully. |
| 7 | Collects required request details and creates the new ID card request. | Executes **createIDCard** tool. | New ID card request created successfully. |
| 8 | Shares confirmation details with the user. | Provides Request ID and expected delivery timeline. | User receives successful confirmation. |
| 9 | User confirms no further assistance is required. | Ends the conversation. | Session terminated successfully. |
